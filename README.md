# House Price Prediction

Predicted sale prices of 1459 houses using 79 features from the Ames, Iowa housing dataset (Kaggle Competition).

## Models Used
Trained and compared Linear Regression (RMSE: 0.1559), Random Forest (RMSE: 0.1442), and XGBoost (RMSE: 0.1339). XGBoost was selected as the best model.

## Features Engineered
Created TotalSF (total square footage), TotalBath (total bathrooms), HouseAge (age of house), and RemodAge (years since remodel).
